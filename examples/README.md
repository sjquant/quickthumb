# Quickthumb Examples

This directory contains example scripts demonstrating various Quickthumb features and use cases.

## Available Examples

### youtube_thumbnail_01.py

Creates a professional YouTube-style thumbnail with:

- Bold headline text with neon green color and black stroke
- Subtitle text
- Bright border
- Background image support (add your own image)
